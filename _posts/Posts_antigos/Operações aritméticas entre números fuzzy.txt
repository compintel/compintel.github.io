Recentemente fiz um post descrevendo os fundamentos da lógica fuzzy, caso não tenha visto, pode ser encontrado <a href="http://www.computacaointeligente.com.br/artigos/fundamentos-da-logica-fuzzy" target="_blank">aqui</a>. Neste post pretendo aprofundar um pouco nas operações aritméticas para números fuzzy triangulares e trapezoidais. Vamos começar definindo os números fuzzy triangulares e logo após os trapezoidais, que possuem uma semelhança muito grande. Além disso, vamos trabalhar com números fuzzy normalizados, o que significa que existe pelo menos um valor pertencente ao [latex]sup_A[/latex] tal que o seu valor de pertinência é igual a 1 (caso esteja falando grego, acesse o post sobre <a href="http://www.computacaointeligente.com.br/artigos/fundamentos-da-logica-fuzzy" target="_blank">fundamentos da lógica fuzzy</a>). Ao final do post, deixo um código em MATLAB de uma implementação para ambos os números.
<h2>Números fuzzy triangulares</h2>
Um número fuzzy triangular [latex]\tilde{a}[/latex] é definido por [latex]\tilde{a} = (a_1, a_2, a_3)[/latex] como mostra a Fig. 1.

[caption id="attachment_204" align="aligncenter" width="305"]<a href="http://www.computacaointeligente.com.br/wp-content/uploads/2015/05/vel_triang.fw_r1_c11.png"><img class="wp-image-204" src="http://www.computacaointeligente.com.br/wp-content/uploads/2015/05/vel_triang.fw_r1_c11.png" alt="" width="305" height="194" /></a> Fig. 1: Número fuzzy triangular[/caption]
<p style="text-align: left;">Observe que para o valor de x igual a [latex]a_2[/latex] o grau de pertinência do mesmo é igual a 1, o que caracteriza o número fuzzy normalizado, e para os demais valores o grau de pertinência varia de acordo com a reta do triangulo no qual o valor pertence. Portanto, para determinar o grau de pertinência para qualquer valor de x, utilizamos:</p>
<p style="text-align: center;">[latex]\mu_{\tilde{a}(x)} =
\left\{\begin{matrix}
0, &amp; x &lt; a_1 \\ \frac{x-a_1}{a_2-a_1},&amp; a_2 \geq x \geq a_1 \\ \frac{x-a_2}{a_3-a_2},&amp; a_3 \geq x \geq a_2 \\ 0, &amp; x &gt; a_3
\end{matrix}\right. [/latex]</p>
<p style="text-align: left;">Com as definições acima, podemos, então, determinar as operações aritméticas de números fuzzy triangulares como sendo:</p>
<strong>Soma:</strong>
[latex] \tilde{a} + \tilde{b} = (a_1, a_2, a_3) + (b_1, b_2, b_3) = (a_1 + b_1, a_2 + b_2, a_3 + b_3)[/latex]

<strong>Subtração:</strong>
[latex] \tilde{a} - \tilde{b} = (a_1, a_2, a_3) - (b_1, b_2, b_3) = (a_1 - b_1, a_2 - b_2, a_3 - b_3)[/latex]

<strong>Divisão:</strong>
[latex] \tilde{a} / \tilde{b} = (a_1, a_2, a_3) / (b_1, b_2, b_3) = (a_1 / b_1, a_2 / b_2, a_3 / b_3)[/latex]

<strong>Multiplicação:</strong>
[latex] \tilde{a}.\tilde{b} = (a_1, a_2, a_3) . (b_1, b_2, b_3) = (a_1 . b_1, a_2 . b_2, a_3 . b_3)[/latex]

<strong>Multiplicação por escalar:</strong>
[latex] k\tilde{a} = k(a_1, a_2, a_3) = (ka_1, ka_2, ka_3)[/latex]

Além das operações decritas acima, podemos definir também a distância entre dois números triangulares. Existem diversas formulas de distância conhecidas. Vamos definir uma usual, conhecida como distância Hamming. Sendo [latex]\tilde{a}[/latex] e [latex]\tilde{b}[/latex] dois números fuzzy triangulares, a distância Hamming entre os mesmos é definida como:
<p style="text-align: center;">[latex] d(\tilde{a},\tilde{b}) = \frac{1}{6}[|(a_1 - b_1)| + |(a_2 - b_2)| + |(a_3 - b_3)|] [/latex]</p>
Por fim, também podemos definir uma forma de <em>rankiar</em> dois números fuzzy, ou seja, comparar qual é maior. Para isso, definimos a expectativa [latex]E[/latex] de um número fuzzy triangular, que nada mais é que um <em>defuzzyficação, </em>ou seja, encontra um número <em>crisp</em> que representa o número fuzzy. Sendo assim, a expectativa é dada por:
<p style="text-align: center;">[latex] E(\tilde{a}) = \frac{a_1 + a_2 + a_3 }{6} [/latex]</p>
Com isso, sendo [latex]\tilde{a}[/latex] e [latex]\tilde{b}[/latex] dois números fuzzy triangulares, [latex]\tilde{a} &gt; \tilde{b}[/latex] se [latex]E(\tilde{a}) &gt; E(\tilde{b}) [/latex], caso contrário, [latex]\tilde{a} &lt; \tilde{b}[/latex]. Por fim, se [latex]E(\tilde{a}) = E(\tilde{b}) [/latex] os dois números são iguais.
<h2>Números fuzzy trapezoidais</h2>
Um número fuzzy trapezoidal [latex]\tilde{a}[/latex] é definido por [latex]\tilde{a} = (a_1, a_2, a_3, a_4)[/latex] como mostra a Fig. 2.

[caption id="attachment_219" align="aligncenter" width="335"]<a href="http://www.computacaointeligente.com.br/wp-content/uploads/2015/05/vel_trap_r1_c1.jpg"><img class=" wp-image-219" src="http://www.computacaointeligente.com.br/wp-content/uploads/2015/05/vel_trap_r1_c1.jpg" alt="Fig. 2: Número fuzzy triangular" width="335" height="213" /></a> Fig. 2: Número fuzzy triangular[/caption]

Observe que para o valor de x igual a [latex]a_1[/latex] e [latex]a_2[/latex] o grau de pertinência do mesmo é igual a 1.

Neste ponto o leitor já deve ter percebido as semelhanças entre o número fuzzy triangular e o trapezoidal. Mais do que isso, já devem ter notado, que o número fuzzy triangular nada mais é do que um número fuzzy trapezoidal degenerado. Experimente fazer [latex]a_1 = a_2[/latex]. Sim, teremos um número fuzzy triangular! Podemos afirmar, então, que o número fuzzy trapezoidal é uma generalização dos dois modelos. A função de pertinência do número fuzzy trapezoidal é descrita por:
<p style="text-align: center;">[latex]\mu_{\tilde{a}(x)} =
\left\{\begin{matrix}
0, &amp; x &lt; a_1 \\
\frac{x-a_1}{a_2-a_1},&amp; a_2 \geq x \geq a_1 \\
1,&amp; a_2 \geq x \geq a_3 \\
\frac{x-a_3}{a_3-a_4},&amp; a_3 \geq x \geq a_4 \\
0, &amp; x &gt; a_4
\end{matrix}\right. [/latex]</p>
De forma sucinta, vamos definir, assim como no triangular, todas as operações aritméticas. Mas de maneira geral, basta acrescentar  [latex]a_4[/latex] nas equações.

<strong>Soma:</strong>
[latex] \tilde{a} + \tilde{b} = (a_1, a_2, a_3, a_4) + (b_1, b_2, b_3, b_4) = (a_1 + b_1, a_2 + b_2, a_3 + b_3, a_4 + b_4)[/latex]

<strong>Subtração:</strong>
[latex] \tilde{a} - \tilde{b} = (a_1, a_2, a_3, a_4) - (b_1, b_2, b_3, b_4) = (a_1 - b_1, a_2 - b_2, a_3 - b_3, a_4 - b_4)[/latex]

<strong>Divisão:</strong>
[latex] \tilde{a} / \tilde{b} = (a_1, a_2, a_3, a_4) / (b_1, b_2, b_3, b_4) = (a_1 / b_1, a_2 / b_2, a_3 / b_3, a_4 / b_4)[/latex]

<strong>Multiplicação:</strong>
[latex] \tilde{a}.\tilde{b} = (a_1, a_2, a_3, a_4) . (b_1, b_2, b_3, b_4) = (a_1 . b_1, a_2 . b_2, a_3 . b_3, a_4 . b_4)[/latex]

<strong>Multiplicação por escalar:</strong>
[latex] k\tilde{a} = k(a_1, a_2, a_3, a_4) = (ka_1, ka_2, ka_3, ka_4)[/latex]

Também vamos definir a distância e o <em>ranking </em>através da expectativa.

<strong>Distância:</strong>

[latex] d(\tilde{a},\tilde{b}) = \frac{1}{8}[|(a_1 - b_1)| + |(a_2 - b_2)| + |(a_3 - b_3)| + |(a_4 - b_4)|] [/latex]

<strong>Expectativa:</strong>

[latex] E(\tilde{a}) = \frac{a_1 + a_2 + a_3 + a_4}{8} [/latex]
O ranking funciona da mesma maneira descrita no triangular.
